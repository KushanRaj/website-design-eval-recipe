# Reports

This folder contains the curated Proximal-facing report artifacts.

## Files

- `proximal-reward-report.pdf`: Reward/metric analysis from the 12-task Claude Code run.
- `proximal-reward-report.html`: HTML source used to render the reward report.
- `task-exhibition-report.pdf`: Small exhibition document showing dataset diversity and agent reproduction behavior.
- `task-exhibition-report.html`: HTML source used to render the exhibition report.
- `assets/`: Derived screenshot panels used by the exhibition report.

The raw run data used to create these reports lives outside this curated report
folder under `harbor-report-data/` in the local workspace. The reports are
self-contained PDFs intended for sharing.
